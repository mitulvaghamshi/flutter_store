# Flutter and Dart Playground Apps

- A Playground for learning Flutter and Dart. This includes dart console app, flutter app, and flutter web app.

## Getting Started

Please follow the instructions in the README file contained in each directory.

## Features

1. Console app
- [fake_world](console/lib/fake_world)
  - Inheritance, Polymorphism, etc.
- [elevation_peak](console/lib/elevation_peak)
  - File IO, math, speed test, etc.
- [checkout_queue](console/lib/checkout_queue)
  - Efficient checkout-queue implementation.

2. Flutter app
- [afonts_app](client/lib/afonts_app)
  - Display asset fonts.
- [ajson_app](client/lib/ajson_app)
  - Parse and display json data using dart pattern matching.
- [ball8_app](client/lib/ball8_app)
  - Magic-8-ball, CustomPainter, etc.
- [gstore_app](client/lib/gstore_app)
  - Simple Google Store with cart and UI.
- [istore_app](client/lib/istore_app)
  - Simple Shopping app UI.
- [sphere_app](client/lib/sphere_app)
  - Simple 3D spherical UI to display 360-degree images on a sphere.

3. Dart Server app
- [server_app](server)
  - Simple server app: static file server, api, database, dart web app etc.
